# 勤怠管理Webアプリ

> 食肉加工業（十勝ロイヤルマンガリッツァ）の営業・加工チーム向けに開発した社内向け勤怠管理システム。  
> **インフラコスト0円**で本番稼働中。

🔗 **[アプリを開く](https://yuhei0723.github.io/kintai/)**

---

## 解決した課題

| Before | After |
|--------|--------|
| 紙・Excelによる手書き打刻 | スマホ・iPadからワンタップで打刻 |
| 月次勤務表を手動で集計（毎月数時間） | GASが毎月1日に自動生成 |
| 有給・代休申請がメール・口頭でバラバラ | 申請〜承認〜履歴をアプリで一元管理 |
| 管理者が個別に連絡して確認 | 管理画面でリアルタイムに全状況を把握 |

---

## 主な機能

### 全社員向け
- **出勤・退勤打刻** — スマホ・iPad対応のレスポンシブUI
- **個人専用URL + QRコード** — 本人しか選択できない誤操作防止設計
- **有給・代休申請** — 申請・変更・取消と残日数のリアルタイム表示
- **申請確認ダイアログ** — 誤送信防止

### パート社員向け
- **シフト希望カレンダー** — 複数日一括・時間帯指定で申請
- **シフト変更・取消申請**

### 管理者向け
- **申請管理ページ** — 有給・シフトをタブで集約、氏名・ステータスでフィルター
- **打刻修正UI** — 時刻修正・削除・追加の3機能
- **月次勤務表の自動生成** — GASが毎月1日に社員別スプレッドシートを自動作成
- **社員管理** — 雇用区分・チーム・有給付与日数・アイコン設定
- **法定休日の自動計算** — 日曜・祝日・土曜・指定休日を区別
- **有給奨励日の全社一括差し引き**

## 技術構成

```
フロントエンド  : HTML / CSS / JavaScript（単一ファイル）
バックエンド    : Google Apps Script（REST API）
データストア    : Google スプレッドシート
ホスティング    : GitHub Pages（無料）
通信方式        : fetch API（CORS対応）
```

**選定理由：** 10名規模の社内ツールとして、追加インフラコストなしに本番運用できる構成を選択。  
管理者がスプレッドシートを直接確認・編集できるため、システム障害時のフォールバックとしても機能する。

---

## 工夫した点

- **役割別UI制御** — パートのみシフト申請を表示、正社員は非表示など雇用区分でUI分岐
- **iPadのタッチ操作対応** — イベントデリゲーション方式で安定動作
- **モーダル重複表示バグ対策** — 全モーダルをbody直下に配置
- **個人URL設計** — アクセスすると氏名が自動セットされ他人の名前を選択不可
